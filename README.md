# SyndicatePro

Real estate syndication waterfall calculator with standardized deal terms.

## What It Does

Takes syndication deal parameters and calculates investor distributions using industry-standard waterfall structures:

- **Simple Split** - Pro-rata distributions
- **Pref + Split** - Preferred return hurdle, then promote
- **Catch-Up** - Full waterfall with GP catch-up provisions

## Current Features

✅ Interactive waterfall calculator
✅ Three waterfall structure types
✅ Investor tracking
✅ Real-time distribution visualization
✅ IRR/multiple calculations
✅ LP/GP split modeling

## Deal Input Variables

| Variable | Description |
|----------|-------------|
| Total Project Cost | Full acquisition + rehab cost |
| Loan Amount | Debt financing |
| GP Equity % | Sponsor's equity contribution |
| Preferred Return | LP hurdle rate (typically 6-10%) |
| GP Promote | GP's share above hurdle (typically 20-30%) |
| Catch-Up Rate | GP catch-up percentage |
| Hold Period | Expected investment duration |
