# Project Roadmap: SyndicatePro

## Vision
A complete "do-everything" platform for real estate syndicators — from deal sourcing to exit distributions.

---

## Phase 1: Foundation (Waterfall Engine) ✅ COMPLETE

**Goal:** Build the core calculation engine that differentiates this from generic tools.

- [x] Standardized deal input variables
- [x] 3 waterfall structure types (Simple, Pref+Split, Catch-Up)
- [x] Real-time distribution visualization
- [x] IRR/Multiple calculations
- [x] Investor tracking

## Architecture Evolution

**Current:** Single HTML file (MVP)
**Phase 2-3:** Add JSON export/import for deal persistence
**Phase 4+:** Backend with database (Firebase/Supabase to start)
**Enterprise:** Self-hosted option with full data control

---

## Monetization Options

1. **SaaS Subscription** - Monthly per deal or per GP
2. **Per-Deal Pricing** - Pay when you close
3. **Document Marketplace** - Premium templates
4. **Services Layer** - Bookkeeping, tax prep add-ons

---

## Competitive Differentiation

vs. **Juniper Square:** More flexible waterfall modeling, document generation built-in
vs. **Generic calculators:** Standardized variables reduce user error
vs. **Law firms:** 10x cheaper, faster turnaround on docs
